# Parallax — 문서

[English](README.md) · **한국어** · [中文](README.zh.md)

Parallax는 local-first 코드 impact 분석 계층이다 — 단일 SQLite 저장소가 CLI, 코딩 에이전트용 MCP 서버, UI explorer를 구동한다. 이 인덱스는 `docs/`의 모든 문서, Claude Code / Codex skill, 아키텍처 심화를 링크한다.
